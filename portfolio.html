Lumetrix Labs

For colleges and institutions to run their own coding-assessment platform. Teachers create classrooms, set assignments, and build full courses — video and interactive — while an automated judge grades every submission. Students get auto-generated records of their work, saving hours of manual record-keeping on both sides. Currently in beta with select institutions.

IN BETA · INSTITUTIONS

Lumetrix Learn

A marketplace where course creators sell real courses, not just videos — built-in code evaluation, a dynamic page for each student tracking their journey through the course, community support per session, and AI-assisted hints and explanations along the way. Currently in beta with select institutions.
